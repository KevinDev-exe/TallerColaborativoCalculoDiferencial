# Graficador de Cálculo Diferencial

Este proyecto permite ejecutar un script en Python para generar **gráficas utilizadas en cálculo diferencial** usando librerías matemáticas.

## Requisitos

* Python instalado
* Terminal (CMD o PowerShell)

---

# Paso 1: Crear el entorno virtual

Ejecuta el siguiente comando en la carpeta del proyecto:

```
python -m venv calculoDiferencial
```

Esto creará un entorno virtual llamado **calculoDiferencial** donde se instalarán las dependencias del proyecto.

---

# Paso 2: Activar el entorno virtual

Dirígete a la carpeta del entorno virtual y ejecuta el script de activación.

## En CMD

```
calculoDiferencial\Scripts\activate
```

## En PowerShell

```
.\calculoDiferencial\Scripts\Activate
```

Si se activó correctamente verás algo así en la terminal:

```
(calculoDiferencial)
```

---

# Paso 3: Instalar las dependencias

Una vez activado el entorno virtual, instala las librerías necesarias:

```
pip install numpy matplotlib
```

Estas librerías permiten realizar cálculos matemáticos y generar gráficas.

---

# Paso 4: Ejecutar el programa

Ubícate en la carpeta del proyecto y ejecuta:

```
py "taller python- calculo fifenrencial.py"
```

El programa generará las gráficas correspondientes al taller de cálculo diferencial.

---

# Tecnologías utilizadas

* Python
* NumPy
* Matplotlib
